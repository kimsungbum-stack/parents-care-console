# 라운드 0: 경쟁사 벤치마킹

## 분석 대상 6개 서비스

### 1. 핑거세일즈 (fingersales.com)
- **분류:** B2B 영업 CRM
- **핵심:** 고객 발굴→매출 전과정 관리, 모바일 지원, AI 기능 (2024.09~)
- **가격:** 월 3만원/인 (프로), 5만원/인 (엔터프라이즈), 무료 플랜 없음
- **가져올 것:** 모바일에서 고객 등록/수정 가능한 UX, 관리자 대시보드 한눈에 보기
- **안 할 것:** 인당 과금 (1인 센터장에겐 부담), B2B 세일즈맨 대상 복잡한 UI, 600+ 기업 고객 대상 기능 과잉

### 2. 핵클 (hackle.io)
- **분류:** A/B테스트 + 데이터분석 + CRM마케팅
- **핵심:** 기능플래그, MAB 테스트, 카카오 알림톡/인앱 메시지
- **가져올 것:** "클릭 몇 번으로 데이터 시각화" 철학, 카카오 알림톡 연동 (장기 로드맵)
- **안 할 것:** 개발자 SDK 중심 온보딩, A/B테스트/기능플래그 등 과잉 기능

### 3. 플레어레인 (flarelane.co.kr)
- **분류:** 푸시알림 + CRM마케팅 자동화
- **핵심:** 다중 채널 (푸시, 카카오, 문자, 팝업), 고객 여정 자동화, 3일 연동
- **가져올 것:** "3일 만에 연동 완료" 빠른 시작 메시지, 고객 여정 자동화 개념 (상태별 리마인더)
- **안 할 것:** 억단위 데이터 처리, 멀티채널 캠페인 복잡성, 개발자 연동 필수 구조

### 4. 채널톡 (channel.io)
- **분류:** 실시간 채팅 상담 + CRM
- **핵심:** 무료 채팅, 프로필봇 자동 정보수집, 카카오/네이버/인스타 연동 무료
- **가격:** 무료(기본 채팅) → MAU 기반 유료 (기능 차이 없이 용량만 확대)
- **가져올 것:** 무료→유료 자연스러운 전환 모델, 프로필봇 = "입력 없이 정보 확보" 패턴
- **안 할 것:** 웹 위젯 중심 (장기요양은 전화 90%), MAU 기반 복잡한 과금

### 5. 노션 (notion.com)
- **분류:** 범용 생산성 도구 → CRM 커스터마이징
- **핵심:** 자유로운 DB 설계, 소규모 팀에 인기, 다양한 CRM 템플릿
- **가져올 것:** "정말 필요한 것만, 단순 반복 업무 위주" 설계 철학, 최소한의 학습 곡선
- **안 할 것:** 직접 구축 필요 (비개발자 불가), 관계형 DB 개념 이해 요구, 알림/리마인더 없음

### 6. 토스 포스 (toss.im)
- **분류:** 소상공인 POS + 고객관리 통합
- **핵심:** 결제 시 자동 정보 수집, 리마인드 알림톡, 맞춤 쿠폰, 무료 고객관리
- **가져올 것:** "결제 시 자연스럽게 정보 수집" = "통화 시 AI가 자동 수집" 동일 패턴, 극도로 단순한 UI
- **안 할 것:** 오프라인 매장 중심 (도메인 다름), POS 종속적 구조

## 포지셔닝 결론

| 경쟁자 | 관계 | 우리의 차별점 |
|--------|------|-------------|
| 핑거세일즈/핵클/플레어레인 | 비경쟁 (다른 타깃) | 장기요양 도메인 특화, 비개발자 대상 |
| 채널톡 | 간접 경쟁 | 전화 문의 중심 vs 온라인 채팅 중심 |
| 노션 CRM | 직접 대체 | "노션보다 쉽고, 장기요양에 특화" |
| 토스 포스 | UI/UX 벤치마크 | "토스처럼 쉬운 CRM" |

## 핵심 인사이트 → 적용 방향

1. **극단적 단순함** (토스 벤치마크) → 주요 작업 클릭 3번 이내
2. **무료→유료 자연 전환** (채널톡 모델) → 무료 체험 충분히, 가치 체감 후 결제
3. **빠른 시작** (플레어레인 "3일") → 온보딩 3분, 첫 가치 체감 10분
4. **입력 부담 제거** (업계 공통 트렌드) → AI 녹음분석 = 킬러 피처
5. **카카오 연동** (업계 표준) → 로드맵에 카카오 알림톡 리마인더

## 절대 따라하지 말 것

1. SDK/개발자 연동 요구 → 타깃이 비개발자
2. MAU/사용량 복잡한 과금 → 센터장은 단순 정액제 선호
3. 대시보드에 차트 10개 → 정보 과부하 = 이탈
4. 영어 UI 용어 → "CRM", "Pipeline" 대신 한국어
5. 멀티채널 캠페인 → 전화+카톡이면 충분
